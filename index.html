Q-Learning-Algorithm

Example of Q-Learning algorithm based on Javascript and jQuery. Given a maze with positive goals and negative goals. The purpose of this algorithm is to find shortest path to the positive goals without knowing anything about blocking tiles, positive goals, and negative goals coordinate. For more explaination about Q learning please visit this page : https://en.wikipedia.org/wiki/Q-learning
